## Tier List Templates

Templates reutilizáveis de Tier List em desenvolvimento.

O comando `/tier template criar` abre um editor com botões para adicionar itens
de texto, URL, avatar, Wikipedia e Spotify, remover/reordenar itens, gerar
preview e publicar a versão atual. O comando `/tier template usar` cria uma
sessão individual a partir da versão publicada, com seleção de item, destino,
paginação, reset e finalização.

Imagens são baixadas, validadas, convertidas para WEBP quando possível e salvas
localmente antes de entrarem no template. Sessões renderizam apenas a partir dos
assets locais.
