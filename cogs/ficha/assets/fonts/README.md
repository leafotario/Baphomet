# Fontes da ficha

as fontes globais do projeto.

- `cogs/ficha/assets/fonts/Poppins-Bold.ttf`
- `cogs/ficha/assets/fonts/Montserrat-Black.ttf`
- `assets/fonts/Poppins-Regular.ttf`
- `assets/fonts/Poppins-Bold.ttf`
