# Regras do Projeto

- **Auto Commit e Push:** Sempre que realizar alterações de código ou adicionar novos recursos a pedido do usuário, execute automaticamente um comando no terminal para commitar (`git add . && git commit -m "..."`) e enviar as alterações para o repositório remoto (`git push`). Não é necessário perguntar se deve commitar e realizar o push; apenas faça-o ao concluir as modificações do turno.
